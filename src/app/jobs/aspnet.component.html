ASP.Net fejlesztő

  • A feladatok, amelyek Rád várnak:

  • ASP.Net alkalmazások fejlesztése
  • adatbázisok tervezése, fejlesztése
  • fejlesztői csapatmunkában történő aktív részvétel
  • saját fejlesztési feladatokkal kapcsolatos adminisztratív és dokumentációs feladatok ellátása

  • Téged keresünk ha:

  • van legalább 2-3 éves .NET fejlesztői tapasztalatod
  • ismered az alábbi Web technológiákat:
    • HTML (HTML5 ismerete előny)
    • CSS (CSS3, LESS ismerete előny)
    • JavaScript, JQuery könyvtár tapasztalat
  • XML, XPath, XSD, XSL, (JSON ismerete előny)
  • legalább írásban és dokumentáció olvasás/megértés szintjén tudsz angolul
  • pontos, alapos és igényes vagy a munkavégzésedre
  • motivált, proaktív hozzáállással végzed a feladataid

  • Szuper, ha rendelkezel még:

  • adatbázis tervezési ismeretekkel
  • adatbázis szerverek, SQL nyelv ismerettel (MS SQL, Oracle)
  • Hibernate köztes réteg ismerettel
  • van szakirányú felsőfokú végzettséged
  • JSP, további Java Web Framework ismereted is van (pl. Spring, JSF, Struts)

  • Amit cserébe mi kínálunk Neked:

  • folyamatos szakmai támogatást és képzési lehetőségeket
  • fejlődési- és karrierlehetőséget
  • jó hangulatú, összetartó, dinamikus csapatban való Budapesti vagy Székesfehérvári munkavégzést
